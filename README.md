# MuCoreX

**Descripción:**  
MuCoreX es un sistema web companion para servidores MU, que funciona junto a cualquier CMS existente. Permite gestionar usuarios, sesiones y addons web de manera segura y centralizada, con una experiencia de usuario fluida a través de una SPA interna.

**Características principales:**  
- Login seguro con token único por usuario.  
- Interfaz web rápida y moderna (SPA).  
- Soporte para addons independientes, sin alterar el CMS principal.  
- Compatible con cualquier CMS MU como sistema acompañante.  

**Cómo usarlo:**  
1. Abrir `index.html` en el navegador.  
2. Iniciar sesión con tu usuario y contraseña de servidor MU.  
3. Explorar y utilizar los addons disponibles.
